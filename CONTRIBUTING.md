# Contributing to SysMon

Thanks for your interest!

## Reporting Bugs
- Include OS and bash version
- Provide command and error
- Show `sysmon` output if relevant

## Suggesting Features
- Explain the use case
- Consider performance impact

## Pull Requests
1. Fork repository
2. Create feature branch
3. Test thoroughly
4. Follow code style
5. Update README
